Privacy Policy

Last updated: May 2026

1. Overview

This extension helps users clean browsing data such as cookies, cache, and local storage on selected websites.

2. Data Collection

This extension does not collect, store, or transmit any personal data to external servers.

3. Permissions Usage

  • cookies / browsingData: delete local browsing data
  • history: remove browsing history locally
  • storage: save user preferences
  • tabs / webNavigation: detect active tab
  • scripting / host permissions: apply cleaning actions

4. External Communication

No external servers are used.
